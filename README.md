# 天意决策 - 易经占卜助手

> 把犹豫交给天意

一款基于《易经》六十四卦体系的现代化决策辅助工具，面向年轻用户群体，提供类似"答案之书"的体验。

## 快速开始

### 本地运行
```bash
cd yijing-decision
python3 -m http.server 8000
```
访问 http://localhost:8000

### 在线体验
直接打开 `index.html` 文件

## 核心功能

✨ **铜钱起卦**：传统铜钱起卦法，投掷6次生成卦象
🎯 **卦象解读**：64卦完整数据，现代化语言解读
🎨 **精美设计**：紫色渐变UI，流畅动画效果
📱 **响应式布局**：支持桌面端和移动端
⚠️ **免责声明**：显著提示"仅供参考"

## 使用流程

1. **输入问题**（可选）- 在输入框中写下你的问题
2. **开始起卦** - 点击"开始起卦"按钮
3. **投掷铜钱** - 连续点击"投掷铜钱"6次
4. **查看结果** - 查看卦象、卦辞、解读和建议
5. **重复使用** - 点击"再占一卦"重新开始
